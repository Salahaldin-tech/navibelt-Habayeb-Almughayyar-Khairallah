## Project Overview

NaviBelt is a Python-based navigation assistance system for the visually impaired.

---
---
## Project Features

- Generate sample navigation routes
- Generate obstacle event logs
- Load and manage route information
- Record obstacle detection events
- Analyze obstacle directions
- Visualize navigation data
